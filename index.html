Voice-Based Parkinson’s Disease Detection

AI-powered early diagnosis using machine learning on voice features

About Project

This project focuses on developing a voice-based Parkinson’s Disease detection system that uses machine learning algorithms to analyze speech features and detect early signs of Parkinson’s.

Methods

  • Data Collection: Voice recordings of patients and healthy individuals.
  • Feature Extraction: Acoustic features such as jitter, shimmer, and pitch.
  • Model Training: Using algorithms like SVM, Random Forest, and Neural Networks.
  • Validation: Cross-validation and accuracy assessment.

Results

The model achieved an accuracy of 85% in distinguishing Parkinson’s patients from healthy controls based on voice features, showing promise for non-invasive early diagnosis.

Publications

  • Shahi, Y.B., et al. “Voice-Based Machine Learning Model for Early Diagnosis of Parkinson’s Disease,” Biomedical Engineering Journal, 2024.
  • Shahi, Y.B., “Speech Signal Processing for Parkinson’s Detection,” International Conference on Biomedical Technologies, 2023.
